How to Build Each Slide in Canva
  • S1
    Hook slide: Navy (#1B2F4E) background. Logo centered-top. "WESLEY CHAPEL" eyebrow in white 30% opacity. Big ALL CAPS headline — key word(s) in sky blue, rest in white.
  • S2–4
    Location slides: Full-bleed photo background. Add a dark gradient overlay (black, 0% opacity top → 80% opacity bottom). Logo top-left, 80px. Location name in sky blue ALL CAPS bold. One-line description in white below.
  • S5
    CTA slide: Navy background. Logo centered. "WHICH ONE IS YOUR FAVORITE?" in white ALL CAPS bold. "Drop it in the comments below" in white 40% opacity below.
Pro tip: Build all 5 slides once as a Canva template. Each day, duplicate the template, swap the photos and headline text only. Everything else stays the same. Should take under 10 minutes.
Where to Get Photos
  • 01
    Google Images: Search "[location name] Wesley Chapel FL" → Tools → Usage rights → Creative Commons. Fast, usually enough.
  • 02
    Unsplash.com: Free high-res photos. Good for generic nature/outdoor spots.
  • 03
    The business's own Instagram: Screenshot and credit them in the caption — they'll usually appreciate the tag.
  • 04
    Ohm's footage: If a spot has been visited, pull a still frame from video.
