How to Build in Canva (One Time)
  • 01
    New design → Custom size → 1080 × 1350 px
  • 02
    Add a full-bleed image placeholder (top 55%). Lock it as a background layer so photos swap easily.
  • 03
    Add a solid navy rectangle (#1B2F4E) covering the bottom 45%. No gradient needed.
  • 04
    At the photo/panel border: add two horizontal lines in white (50% opacity) with "GODDARD NEWS" text centered between them. Group these together.
  • 05
    Add the Goddard logo top-left. Put a white circle behind it if it doesn't pop on the photo.
  • 06
    Inside the navy panel: location tag (small, muted), location name in Anton + yellow, one-line description in white.
  • 07
    Duplicate this slide 3 times (for the 3 spots). Build hook and CTA slides separately — navy background only, no photo.
  • 08
    Save as a template. Every morning: duplicate, swap the photo, paste the new text. Done in under 10 minutes.
Daily workflow: Say "run the carousel" → I give you 3 story options → you pick one → I output all 5 slides copy-paste ready → open Canva template → swap photo + paste text → post.
Where to Get Photos
  • 01
    Google Images: Search "[place name] Wesley Chapel FL" → Tools → Size: Large. Right-click → Save. Fast and usually enough.
  • 02
    The business's own Instagram: Screenshot a good photo and tag them in the caption — they'll appreciate it and often reshare.
  • 03
    Unsplash.com: Free high-res photos for generic outdoor/nature spots.
  • 04
    Ohm's footage: Pull a still frame from video if a spot has been visited.
